# 休息大过天 - 安卓App规格文档

## 1. 项目概述
- **项目名称**: 休息大过天
- **类型**: Android原生应用
- **核心功能**: 记录每月请假天数（3天=6个半天），点击格子选择休假日期上午/下午

## 2. UI/UX规格

### 布局结构
- 单页面应用
- 顶部标题区域
- 6个等大格子（2行3列或3行2列网格）
- 底部提示文字

### 视觉设计
- **主色调**: 
  - 背景: #F5F5F5 (浅灰)
  - 格子默认: #4CAF50 (绿色，代表可用假期)
  - 格子已用: #BDBDBD (灰色，暗淡)
  - 文字: #212121 (深灰)
- **字体**: 
  - 标题: 24sp, 加粗
  - 格子文字: 18sp
  - 提示: 14sp
- **间距**: 16dp 边距，12dp 格子间距
- **圆角**: 8dp

### 组件
- **可用格子**: 绿色背景，白色文字，显示"请假"
- **已用格子**: 灰色背景，黑色文字，显示具体日期如"4月5日 上午"

### 交互
- 点击可用格子 → 弹出日期选择对话框
- 对话框包含: 日期选择器 + 上午/下午单选按钮
- 确认后格子更新样式和文字

## 3. 功能规格

### 核心功能
- 显示6个半天假期格子
- 点击格子选择具体日期（1-30日）+ 时段（上午/下午）
- 持久化存储（SharedPreferences）
- 重置功能（月底手动重置）

### 数据存储
- 已用假期列表: 日期+时段
- JSON格式存储在SharedPreferences
